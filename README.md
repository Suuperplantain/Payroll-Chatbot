# Payroll Pilot Assistant

Payroll Pilot Assistant is a payroll support chatbot project designed to help employees get quick answers to common payroll-related questions. The system is intended to provide support with things such as payslip information, payroll guidance, employee requests, and HR-related escalation when needed.

This project is being developed as a prototype to demonstrate how a payroll chatbot can improve access to payroll support through a simple and user-friendly interface.

## Project Status

This project is currently in progress.
